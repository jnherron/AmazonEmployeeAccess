# AmazonEmployeeAccess
Amazon Access Kaggle Competition

AmazonAnalysis.R
- Loads in data
- Basic EDA
- Feature engineering / creates recipe
- Performs Logistic Regression
- Performs Pentalized Logistic Regression
- Random Forest
- KNN Model
- Naive Bayes
- Neural Network
- SVN (Polynomial, Radial, and Linear)

LocalAnalysis.R and ServerAnalysis.R were created while testing which of the
models and recipe combinations had the best predictive power. They do not provide
any knew analysis, they were simply "scratch paper" where I could copy, paste,
and delete code during the testing process.
